Dictionary

In Python dictionaries are unordered, changeable and indexed collections.

Get Key and Value

a = {"1":1, "2":2, "3":3}
a.items()

Get All Keys

a = {"1":1, "2":2, "3":3}
b = {"2":2, "3":3, "4":4}
a.keys()

Find Same Keys

a = {"1":1, "2":2, "3":3}
b = {"2":2, "3":3, "4":4}
c = set(a).intersection(set(b))
list(c)

Update Dictionary

a = {"1":1, "2":2, "3":3}
b = {"2":2, "3":3, "4":4}
a.update(b)
a

Merge Dictionaries

a = {"x": 55, "y": 66}
b = {"a": "foo", "b": "bar"}
c = {**a, **b}
c
